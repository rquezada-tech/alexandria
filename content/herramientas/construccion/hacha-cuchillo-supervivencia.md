# Hacha y Cuchillo de Supervivencia — Fabricación y Uso

El cuchillo y el hacha son las dos herramientas fundamentales de supervivencia. Todo lo demás depende de estas dos. Saber fabricarlas, afilarlas y usarlas correctamente es la habilidad más importante del kit de supervivencia.

## El cuchillo ideal de supervivencia

Un cuchillo de supervivencia debe ser:
- Fijo (no plegable, para máxima solidez)
- Hoja de 10-15 cm
- Acero al carbono (alto carbono) o acero inoxidable
- Espesor de hoja mínimo 3mm
- Mango ergonómico que no resbale con agua
- Funda rígida y resistente al agua

Materiales recomendados:
- Acero 1095 (alto carbono, fácil de afilar)
- Acero 440C (inoxidable, más duro)
- Acero 154CM (premium, mantiene filo más tiempo)
- Titanium (ligero, pero más caro y difícil de trabajar)

## Fabricación de cuchillo con hoja de sierra

Un cuchillo funcional puede fabricarse a partir de una hoja de sierra vieja:

Materiales:
- Hoja de sierra de mano (acero 1095 aprox.)
- Madera dura o micarta para el mango
- Pegamento epoxy industrial
- Cuerda de paracord para envuelto

Proceso:
1. Traza el perfil del cuchillo sobre la hoja de sierra
2. Corta con amoladora angular (angle grinder), enfriando constantemente con agua
3. Perfora los agujeros del mango
4. Lija y afila el filo con piedra de afilar
5. Fabrica las escalas del mango y átalas con paracord
6. Trata la hoja con aceite para prevenir óxido

## Hacha de emergencia con piedra

El hacha más primitive usa una piedra como cabeza:

Materiales:
- Piedra dura (basalto, cuarzo, jade artificial)
- Madera de densidad media (roble, fresno, avellano)
- Cuerda de fibras vegetales o tendon animal
- Mazo para ajustar

Proceso:
1. Selecciona una piedra con un borde cortante natural
2. Busca un palo curvo (~60cm) que sirva como mango
3. Practica una muesca en el palo donde descansará la piedra
4. Fija la piedra con cuerdas apretadas en múltiples direcciones
5. Refuerza con resinas naturales o cera de abeja

Esta hacha es primitiva pero funcional para cortar madera blanda.

## Hacha de tubo de cobre

Una hacha funcional puede fabricarse con un tubo de cobre:

Materiales:
- Tubo de cobre de 22mm (agua)
- Disco de carbide o piedra de afilar
- Palo de avellano o similar (~80cm)
- Cuerda de paracord
- Epoxy de dos componentes

Proceso:
1. Corta el tubo de cobre a 15cm
2. Rellena el tubo con epoxy mezclado con arena (para peso)
3. Inserta el mango de madera y fija con cuerda bien apretada
4. El borde del tubo de cobre no es muy cortante, pero funciona para trabajos livianos

## Piedras de afilar

Toda herramienta cortante necesita una piedra de afilar:

Tipos de piedras:
- Carburo de silicio (verde, grano 100-220): para afilado gross
- Oxido de aluminio (marron, grano 400-600): para afilado medio
- Arkansas (piedra natural, grano 1000-8000): paraafilado fino
- Diamante (placa metalica con diamantes): muy duradera

El orden correcto: gross → medio → fino
Siempre afilar perpendicular al borde (mismo ángulo que el filo original)

## Sierra de cable

Una sierra de cable puede fabricarse para cortar huesos o madera gruesa:

Materiales:
- Cable de acero (cable de freno de bicicleta funciona)
- 2 mangos de madera (palos cortos)
- Alambre para asegurar los mangos

Proceso:
1. Corta el cable a ~1 metro
2. Fija cada extremo a un mango de madera con alambre
3. Los mangos deben tener un agujero para asegurar el cable
4. Tensa el cable y usa movimiento de vaivén

El borde dentado del cable de freno funciona como dientes de sierra.

## Hacha de piedra con bisagra

Método avanzado para mayor solidez:

Materiales:
- Bisagra de puerta grande (hierro fundido)
- Mango de madera dura
- Cuna de madera para la cabeza
- Epoxy + vendas de fibra de vidrio

Proceso:
1. La cabeza de la bisagra tiene un eje que sirve como oído
2. Fabrica el mango para que entre en el oído
3. Sella con epoxy yenvuelvecon fibra de vidrio
4. Afila el borde inferior de la bisagra

## Mantenimiento del filo

Reglas fundamentales:
1. Nunca afilar en sentido contrario al uso (solo un lado del movimiento)
2. Mantener ángulo constante durante todo el afilado
3. Usar cuero para honing (borrador de barbería) como paso final
4. Proteger contra óxido: limpiar, secar, aceitar

Ángulos de afilado según uso:
- Cuchillo de cocina: 15-20°
- Cuchillo de supervivencia: 20-25°
- Hacha: 25-30°
- Sierra: 45° (dientes apenas)

## Funda para cuchillo

Una funda protege el filo y previene accidentes:

Materiales mínimos:
- Cuero grueso o canvas pesado
- Hilo de nylon resistente
- Agujas de cuero o machine de coser

Tipo básico de funda:
1. Corta dos piezas de cuero en forma de U
2. Coser los bordes con costuras dobles
3. Hacer presilla para el cinturón
4. Sellar los bordes con fuego o goma

También puede fabricarse una funda de Kydex (plástico termomoldeable) con una hoja de polytetafluoroetileno.
