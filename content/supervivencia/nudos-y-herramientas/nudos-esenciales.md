# Nudos Esenciales y Herramientas Improvisadas

Un nudo mal hecho ha matado a más personas en emergencias que la mayoría de los otros factores combinados. No es un detalle menor: es una decisión de ingeniería. Esta guía cubre los nudos que真正 importan en supervivencia — los que puedes hacer con manos frías, torpes, en la oscuridad, bajo estrés.

> **Regla número uno de los nudos**: si no puedes desatarlo con las manos libres, no lo ates bajo carga. Un nudo que parece seguro pero no se puede soltar es una trampa.

---

## Nudos de amarre (Hitches)

Los nudos de amarre sirven para fijar una cuerda a un objeto: un poste, un árbol, una argolla.

### 1. Clove Hitch (Nudo Clavo)

El más básico y versatile. Se deshace tirando de un extremo.

```
//images/nudos-01-amarre.svg
```

**Cómo se hace:**
1. Pasa la cuerda alrededor del poste
2. Cruza por encima del tramo largo
3. Pasa nuevamente alrededor del poste
4. Introduce el extremo bajo la última vuelta

**Ventaja**: se deshace rápidamente. **Desventaja**: puede deslizarse bajo carga lateral.

### 2. Taut-line Hitch (Nudo Ajuste)

El nudo de las tiendas de campaña. Se desliza para ajustar tensión pero no vuelve solo.

```
//images/nudos-01-amarre.svg
```

**Cómo se hace:**
1. Ata un clove hitch alrededor del poste
2. Da dos vueltas con el extremo alrededor del tramo largo, por encima
3. Termina con un medio nudo dentro del bucle

**Uso**: ajustar la línea de una tienda, tender ropa, sujetar un toldo.

### 3. Timber Hitch (Nudo de Arrastre)

Para arrastrar madera o postes. La carga longitudinal tira del nudo haciéndolo más firme.

```
//images/nudos-01-amarre.svg
```

**Cómo se hace:**
1. Haz una vuelta larga alrededor del objeto
2. Cruza por encima del tramo largo
3. Gira 3-4 veces por encima de tu propia cuerda (espiral)
4. Tira fuerte

**Uso**: arrastrar leña, mover postes, halar tuberías.

### 4. Killick Hitch (Nudo de Levantamiento)

Para elevar piedras u objetos redondos sin que resbalen.

```
//images/nudos-01-amarre.svg
```

**Uso**: construir un mortero de piedra, levantar anclas, mover rocas.

## Nudos de unión (Bends)

Los bends unen dos cuerdas. La regla de oro: **la cuerda más delgada forma el bucle, la más gruesa pasa por dentro**.

### 5. Sheet Bend (Cola de Golondrina)

El más usado para unir cuerdas de distinto grosor.

```
//images/nudos-02-union.svg
```

**Cómo se hace:**
1. Dobla la cuerda DELGADA formando un bucle
2. La cuerda GRUESA pasa POR DENTRO del bucle desde abajo
3. La gruesa cruza sobre el extremo libre de la delgada
4. Tira de ambas partes opuestas

**Si inviertes la orientación**, bajo carga el nudo puede deslizarse. Recuerda: **delgada = bucle, gruesa = por dentro**.

### 6. Double Sheet Bend (Sheet Bend Reforzado)

Cuando las cuerdas están mojadas, húmedas o muy resbaladizas.

```
//images/nudos-02-union.svg
```

**Diferencia**: la gruesa da dos vueltas dentro del bucle en vez de una. Mucho más seguro en condiciones húmedas.

### 7. Carrick Bend (Cabo Marítimo)

Para cuerdas gruesas y húmedas. Muy resistente, casi imposible que se atasque.

```
//images/nudos-02-union.svg
```

**Uso**: navegación, sogas de muelle, cuerdas de aceite o embreadas. No se deshace bajo carga cíclica (lluvia/sequía).

## Nudos de lazo (Loops)

Crean un lazo que no se cierra ni se aprieta con la carga.

### 8. Figure-8 (Cola de Ocho)

El stopper knot más versátil. Fácil de identificar, fácil de deshacer, no se atasca.

```
//images/nudos-03-lazos.svg
```

**Uso**: detener el paso de una cuerda por una polea, hacer un final de seguridad, nudo de parada.

### 9. Bowline (Lazo Fijo)

**El nudo más importante de la supervivencia**. No se aprieta bajo carga, se deshace fácil después. También llamado "nudo del cazador" porque sirve para todo.

```
//images/nudos-03-lazos.svg
```

**Cómo se hace (3 pasos):**
1. Forma un bucle con el tramo largo (la "cola")
2. Pasa la cola por dentro del bucle del tramo de carga (la "vena")
3. La cola pasa POR ENCIMA de la vena y vuelve a bajar por dentro del bucle

**Verificación**: deben verse dos partes de la misma cuerda cruzándose en el bucle. Si ves tres partes diferentes, está mal.

**Uso**: asegurar a un árbol, hacer un lazo para el pie, crear un punto de anclaje, asegurar un rescatado.

### 10. Bowline on a Bight (Lazo Doble)

Hace dos lazos con una sola cuerda. Para rappel rápido.

```
//images/nudos-03-lazos.svg
```

**Uso**: rappel, asegurar a dos personas a la vez, crear dos puntos de anclaje independientes con una sola cuerda.

### 11. Alpine Butterfly (Mariposa)

Hace un loop en medio de una cuerda sin necesidad de acceder a los extremos. Imprescindible para crear un punto de anclaje intermediate.

```
//images/nudos-03-lazos.svg
```

**Uso**: dividir una cuerda larga en segmentos, crear punto de anclaje en el medio, retener peso en el centro de una cuerda.

### 12. Prusik (Ascensor)

El nudo que sube por la cuerda bajo el peso del cuerpo y se desliza con las manos. Inventado por los hermanos Austrian Fritz y Heinrich Prusik.

```
//images/nudos-03-lazos.svg
```

**Cómo se hace:**
1. Necesitas una cuerda auxiliar (más fina que la principal)
2. Haz un bucle con la cuerda fina
3. Ata los extremos del bucle con un nudo seguro (commando o double overhand)
4. Envuelve la cuerda principal 3 veces, pasando el bucle por encima cada vez
5. Bajo peso, el nudo se agarra; con las manos, se desliza

**Uso**: ascender por una cuerda fija, crear un punto de retención, arrests en rappel.

## Nudos de tope (Stopper Knots)

Evitan que una cuerda se deslice por una polea o argolla, o simplemente evitan que el extremo se deshaga.

### 13. Overhand Knot (Nudo Simple)

El más básico. Pero es tan pequeño que puede pasar por una polea. Úsalo solo como referencia visual o para detener un extremo.

```
//images/nudos-04-referencia.svg
```

### 14. Stevedore Knot (Nudo del Estibador)

Un overhand más grande, con una vuelta extra. No pasa por las poleas normales.

```
//images/nudos-04-referencia.svg
```

**Uso**: detener cuerdas en poleas, evitar que un extremo se deshaga permanentemente.

## Nudos de escape rápido

### 15. Highwayman's Hitch (Nudo de Escape)

Se deshace con un solo tirón. Para atar algo que necesitas soltar rápido: animales, puertas,tmp securing.

```
//images/nudos-04-referencia.svg
```

**Cómo se hace:**
1. Haz un bucle alrededor del poste
2. Pasa el extremo por dentro del bucle
3. Tira del extremo libre a través de la segunda vuelta
4. Para soltar: un tirón fuerte del extremo libre

**Uso**: atar caballos, asegurar una puerta que necesitas abrir rápido, tender una línea de emergency que pueda cortarse.

## Guía de selección rápida

```
//images/nudos-04-referencia.svg
```

| Situación | Nudo recomendado |
|---|---|
| Atar a un poste | Clove Hitch |
| Ajustar tensión (tienda) | Taut-line Hitch |
| Unir dos cuerdas | Sheet Bend (o Double si mojadas) |
| Lazo que no se aprieta | Bowline |
| Loop en el medio de cuerda | Alpine Butterfly |
| Subir por cuerda | Prusik |
| Detener cuerda en polea | Figure-8 o Stevedore |
| Soltar rápido | Highwayman's Hitch |
| Arrastrar madera | Timber Hitch |
| Levantar piedras | Killick Hitch |
| Cuerdas gruesas/mojadas | Carrick Bend |

## Verificación: la checklist obligatoria

Antes de confiar tu vida a cualquier nudo:

1. **Inspección visual**: ¿las curvas se cruzan correctamente? ¿no hay pellizcos?
2. **Carga gradual**: aplica peso poco a poco, no todo de golpe
3. **Bajo carga máxima**: verifica que no se mueva, no se deslice, no haga ruido
4. **Prueba de liberación**: asegúrate de poder desatarlo sin herramientas
5. **Factor humano**: si estás cansado, angustiado o con frío, hazlo dos veces

> Un nudo que no puedes deshacer con manos frías y torpes no es un nudo de supervivencia. Es una备etan trap.

## Herramientas improvisadas

### Cuchillo de campo

La elección más importante: **acero al carbono (high carbon) sobre acero inoxidable**. El acero inoxidable no se oxida, pero:
- Es más difícil de afilar en campo
- Pierde el filo más rápido
- Es más quebradizo bajo impacto

El acero al carbono se oxida, pero:
- Se afila fácilmente con cualquier piedra
- Mantiene mejor el filo en uso rudo
- Es más fácil de reparar

**Ángulo de afilado**: 15-20° para supervivencia. Más bajo (10°) = filo más delicado. Más alto (25°) = más resistente pero menos cortante.

### Hachas

Un hacha mal balanceada es peligrosa. El centro de gravedad debe estar lo más cerca posible de la cabeza. Para probarla:
1. Apóyala horizontalmente en un dedo
2. El punto de equilibrio debe estar en la cabeza, no en la empuñadura

**Peso de la cabeza**: 1-1.5 kg para trabajo general. Más liviana = más control. Más pesada = más corte por golpe.

### Piedras como martillos

Una piedra lisa de 1-2 kg atada a un tramo de cuerda o soguín es un percussionador válido para romper huesos, partir leña pequeña, o servir como backup de un martillo.
